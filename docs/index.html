作品概要

本作は、緊迫感のある屋内戦をテーマにした3Dタクティカル・シューティングゲームです。
プレイヤーは特殊部隊員となり、最新兵器が製造される敵国の重要拠点へ単身潜入。数多の敵を退け、最深部に待ち構える巨大兵器の撃破を目指します。

Phase 01: 潜入・探索
ミニマップを頼りに、広大な工場内を索敵。敵の配置を把握する戦略性が求められます。

Phase 02: 敵軍掃討
派手な火花や爆発エフェクトを実装し、シューティングならではの爽快感を追求しました。

Phase 03: 最終決戦
巨大なHPバーと燃え盛る炎が彩るボスバトル。プレイヤーの技術が試されるラストシーンです。

まだいろいろなゲームに慣れきっていないような人たちでも楽しめるよう、複雑すぎる機能や要素を減らし誰でも直感的に楽しめるような作品にした。また、少し慣れてきた人やもともとやっている人でも楽しめるよう、ゲームの難易度の調節やクリアタイムを競えるスピードラン要素を加えました。

技術的な工夫・こだわり

1. 独自のアルゴリズム/システムの構築

敵の動きの細分化をし複数条件下での行動変化に注力しました。また,ミニマップの構築やロケットの攻撃などゲーム体験を向上させられる工夫をしました。

2. ユーザーエクスペリエンス（UI/UX）

誰でも簡単にプレイできるように複雑な動きを入れず,wasdとspaceのみの移動操作と攻撃だけにしシンプルにゲームを楽しめるようにしました。 また,リアルタイム更新のミニマップにより敵やプレイヤーの位置を簡単に把握できる点や画面内でのプレイヤーの位置,体力や弾数を一目でわかるよう調節しました。 　 他にもクリアタイムを記録することで何度も挑戦し,ゲーム体験の向上を図りました。

課題と解決

【課題】 敵の動きを作る中で,敵がプレイヤーを見つけるときの条件が壁を挟んで時も反応してしまったりと不自然な動きになることが多かった。

【解決】 まずは一度プレイヤーを見つけるときの条件として現実的に必要なものを書き出し,必要でないものや特定のプログラムを入れることで逆に必要になるものを考え実装しました。 またこれにより必要な情報を必要な分だけ実装することにより無駄なプログラムにより発生するバグやあとで見返した時でもわかりやすく考えられるようになりました。
